- 这是一个 Remotion 视频生产工程。
- 视频生产必须遵循：分镜 → 素材确认 → scene 配置 → 关键帧 QC → 低清预览 → 高清渲染。
- 不允许直接按 SRT 每句话随机堆特效。
- 一段口播只承担一个主要视觉任务。
- 每段最多一个主组件，加零到两个辅助标注。
- 人物居中时，不允许普通组件进入人物安全区。
- 所有模式下，不允许正文信息进入字幕安全区。
- EvidenceClip 等证据类画面必须绑定可追溯来源。
- 没有来源的数据、新闻、网页内容不得伪装成事实证据。
- 无法确认来源时，只能使用抽象解释型视觉。
- 预计未来不能复用至少两次的效果，不得进入公共组件库。
- 一次性效果应留在具体 episode 内。
- 每次任务执行都不要进行深度测试，除非用户明确要求深度测试。
- 每次改动后必须执行相关 lint、typecheck、test 或 smoke test。
