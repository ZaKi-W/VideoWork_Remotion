# Visual System

视觉体系名称：白底科技编辑部 / AI 纸面实验室。

原则：

- 白墙实拍人物与纸白资料底保持统一。
- 人物居中时，普通信息不得进入人物安全区。
- 所有正文信息不得进入字幕安全区。
- 没有可靠来源时，只使用抽象解释型视觉，不伪造证据。
- 公共组件必须至少预计复用两次；一次性效果留在具体 episode 内。
- 叠加在实拍口播上的文字组件默认必须带黑色渐变底或等效暗色 backplate，保证在浅色墙面和运动背景上可读；底色需柔和融入画面，不做硬卡片。

当前保留的通用视觉组件：

- `NarrationEchoLayer`：普通口播的左侧提要式短句层，保留。
- Acid 系列全屏组件：仅用于已经沉淀为可复用风格的全屏主题视觉。

系统预览项：

- `TalkVideoBase`：口播视频、音频和 SRT 字幕底层，不视为组件，不使用 `C` 编号。

本期一次性视觉：

- `RemotionTalkEffect` 仅服务 `episodes/RemotionTalk/`，不进入公共组件库。
